== Description ==

Wordpress plugin that adds Submit to Google Index button to post edit pages. 

Making it easy to submit posts to the "Fetch as Google" portion of Google Webmaster Tools.

 Only works when posts are already published. 

Make sure your site url matches the one registered with Google Webmaster Tools.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click "Submit to Google" on posts that you'd like to submit to the Google index.


== Frequently Asked Questions ==

= Does this plugin submit the post to the Google index automatically? =

No, you must click the button to submit the post to Google yourself. This plugin only provides an easier method for the submission process.

= Can the button be stylized? =

Yes, the button follows the conventions used by the normal "Submit" button in Wordpress and can be customized.
